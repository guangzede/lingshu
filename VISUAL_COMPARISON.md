## 按钮交互增强

### 基础样式
```
优化前:
  padding: 24px 26px
  border: 1px
  box-shadow: 外阴影 + 内光晕

优化后:
  padding: 26px 28px      ✓ 微调增大

  box-shadow:
    - 外阴影: 0 12px 30px rgba(0,0,0,0.4)

    - 高光线: inset 0 0 1px rgba(255,215,0,0.1)  ✓ 新增细节
```

### 高光层伪元素 (新增)
```
.menu-button::before {
  content: '';
  background: linear-gradient(
    180deg, 
    rgba(255,215,0,0.1), 
    transparent
  );
  opacity: 0;  // 默认隐藏
  transition: opacity 0.2s ease;
}

.menu-button:active::before {
  opacity: 1;  // 按下时显示 ✓ 按压反馈
}

效果对比:
  未按: 卡片浮动
  按下: [↓] 高光层显示 + 阴影深化 = 3D 凹陷感
```

### 交互反馈
```
优化前:
  :active { transform: translateY(1px) }

优化后:
  :active { 
    transform: translateY(2px)  ✓ 更明显
    box-shadow: [强化内光晕]    ✓ 视觉反馈
  }

```
